# Inventario-ikt-skd
Mi portafolio de proyectos

Sistema de inventario para organizar y controlar equipos IKT en un entorno escolar.

## Descripción

Este proyecto está pensado para documentar y organizar equipos tecnológicos como PCs, iPads, impresoras, pantallas, teclados, ratones, cables, adaptadores y otros recursos IKT.

## Objetivo

Mantener una visión clara de los equipos disponibles, su ubicación, estado y uso.

## Estado del proyecto

En desarrollo.
